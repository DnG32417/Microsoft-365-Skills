## Incident Workflow
1. Identify suspicious email (user report or alert)
2. Isolate impacted user mailboxes
3. Pull message from all inboxes using Threat Explorer
4. Reset credentials and enforce MFA
5. Investigate sign-in logs and audit trail
6. Mitigate domain spoofing
7. Close incident with remediation notes
