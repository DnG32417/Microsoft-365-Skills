##Identity
- Enforce MFA globally
- Block legacy authentication
- Use Security Defaults or Conditional Access baseline
- Enable Azure AD Identity Protection

## Device
- Require compliant or hybrid Azure AD joined devices
- Enforce BitLocker on endpoints
- Apply device compliance policies

## Data
- Enable DLP policies
- Configure retention labels
- Secure SharePoint and OneDrive external sharing

## Admin
- Use PIM for role activation
- Maintain break-glass accounts with monitoring
- Enable audit logs retention
